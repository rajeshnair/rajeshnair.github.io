Hypothesis function

Hypothesis function tries to provide a model for the training data to predict future values

h(x) = p1 + p2*x

where p1 is dedcued parameter 1 and p2 is parameter 2

Cost function

Cost function is a function to measure the cost of a given hypothesis in order to ascertain the validity of hypothesis. A hypothesis with 0 cost means it perfectly describes the training data for all the data points. Higher the cost means more inaccurate is the hypothesis. Our aim would be to come up with minimize the cost by coming upo with better parameters.

Cost(p1,p2) = 1/2m * sum(sqr(h(x) - y))

or

Cost(p1,p2) = 1/2m * sum(sqr((p1 + p2*x) - y))

where x is independent variable and y is dependent variable
